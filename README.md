# 🚀 InterviewX

AI-powered mock interview platform featuring resume analysis, voice-based interviews, live coding challenges, and intelligent real-time feedback analytics.

---

## ✨ Features

- 🎤 AI-powered mock interviews
- 📄 Resume-based question generation
- 🧠 Intelligent AI feedback system
- 💻 Live coding challenges
- 🔊 Voice-based interview interaction
- 📊 Performance analytics dashboard
- 🕘 Interview history tracking
- 👨‍💻 Multi-role interview support

---

## 🛠️ Tech Stack

### Frontend
- React.js

- CSS

### Backend
- Node.js
- Express.js

### AI & APIs
- Gemini AI
- AssemblyAI

### Clone Repository

```bash

```

---

### Frontend Setup

```bash
cd client
npm install
npm run dev
```

---

### Backend Setup

```bash
cd server
npm install
npm start
```
